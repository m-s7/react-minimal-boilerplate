# React Minimal Boilerplate

A bare minimum React Boilerplate.

#### Includes:
- React 18
- TypeScript
- Webpack5
- ESLint
- Jest
- SCSS (+css modules)

## Installation

### Clone Repository

```
git clone https://github.com/m-s7/react-minimal-boilerplate.git
```

### Install dependencies

```
npm install
```

### Project build

```
npm run build
```

### Project start

```
yarn run dev
```
